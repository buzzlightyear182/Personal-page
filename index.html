Postboard

a Twitter-like application built with Ruby on Rails

I built this Twitter-like application as a self-learning practice to explore more intermediate concepts and using new tools. This includes: complex associations and polymorphism, use of ActiveSupport::Concerns and ActiveModel::Naming. Also a lot of refactoring and clean code implemented.

Gems used: Clearance for User Authentication, Bourbon, Neat & Bitters for Front-end, Paperclip for File Uploads and Sunspot for better Search functionality.
